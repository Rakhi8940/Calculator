**Calculator Web App**
This is a simple calculator web application that allows users to perform basic arithmetic operations such as addition, subtraction, multiplication, division, and percentage calculations. The app features a modern and visually appealing user interface with responsive design to ensure it works across various screen sizes.

**Features**
1. Basic Operations: Addition (+), Subtraction (-), Multiplication (*), Division (/), and Percentage (%).
2. Clear and Delete Functions: Clear the entire input or delete the last input.
3. Responsive Design: Optimized for mobile, tablet, and desktop screens.
4. Modern UI: Clean, colorful interface with hover effects and smooth transitions.
5. Interactive Buttons: Each button reacts to clicks, providing a user-friendly experience.

**Technologies Used**
1. HTML: Structuring the content of the calculator.
2. CSS: Styling the calculator interface, including responsive design features.
3. JavaScript: Implementing the logic for the calculator's operations.

**Usage**
1. Enter Numbers: Click on the number buttons (0-9) to input numbers.
2. Perform Operations: Use the operation buttons (%, /, *, +, -) to enter operations.
3. Clear Input: Click the "C" button to reset the calculator.
4. Delete Last Input: Click the "DEL" button to remove the last entered digit.
5. Calculate Result: Click the "=" button to compute the result of the operation.
6. View Operations: The current operation and operands are displayed at the top of the calculator screen.
