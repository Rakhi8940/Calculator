# Calculator Project

A simple web-based calculator built using **HTML**, **CSS**, and **JavaScript**. This project demonstrates basic web development skills by creating a functional calculator with a responsive design.

## Features

- Basic arithmetic operations: Addition, Subtraction, Multiplication, and Division
- Clear and reset functionality
- Responsive design for mobile and desktop devices
- User-friendly interface with styled buttons and display

## Technologies Used

- **HTML**: For structuring the calculator interface.
- **CSS**: For styling the layout and buttons.
- **JavaScript**: For handling user interactions and performing calculations.
